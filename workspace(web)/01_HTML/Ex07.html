HTML에서 특수 문자 처리하는 방법
- ~보다 작은() : <
- ~보다 큰(>) : >
- Ampersand : &
- copyright : ©
- Registered trademark(등록상표) : ®
- Trademark(상표) : ™
- Quotation mark(이중 인용 부호) : "
- Opening single quote mark(인용부호 시작) : ‘
- Closing single quote mark(인용부호 종료) : ’
- Medium list dot(가운데 점) : ·
- Space(공백) : 공        백
