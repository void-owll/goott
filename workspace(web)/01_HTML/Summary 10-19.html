240119 HTML 2강

  1. <hr>태그
    • 화면에 수평선을 그리는 태그
    • 기본 값으로 가운데 정렬
    • width 속성으로 길이 조절
  2. <font>태그
    • 폰트와 관련된 태그
    • size="1~7"으로 글자 크기 변경
    • size가 클 수록 큰 글자로 표시됨
  3. text 서식과 관련된 태그들
    • <b> : 굵게 표시
    • <i> : 기울임꼴 표시
    • <small> : 글자를 작게 표시
    • <sub> : 아랫첨자
    • <sup> : 윗첨자
    • <ins> : 밑줄
    • <del> : 취소선
  4. 목록 태그들
    • 목록 선언 후 아이템을 내부에 추가하는 식으로 활용
    • <ul>(unordered list) : 순서가 없는 리스트
    • type=square, disc, circle 등으로 모양 변경
    • <ol>(ordered list) : 순서가 있는 리스트
    • type=1,a,A,i,I 등으로 순서 표시 변경
    • <ul>
          <li>
              아이템
          </li>
      • </ul>
  5. 정의형 목록
    • 용어를 정의하고 설명하는 태그
    • <dl> : 정의형 목록의 선언
    • <dt> : 정의할 용어
    • <dd> : 정의(용어의 뜻)
    • <dl>
          <dt>용어</dt>
          <dd>설명</dd>
      • </dl>
  6. <img>태그
    • 이미지를 추가하는 태그
    • src="이미지 경로" (필수)
    • alt="대체 문구" (권고) : 이미지를 표시할 수 없을 때의 대체 텍스트
    • title="텍스트" : 이미지에 마우스를 올렸을 때의 설명
  7. <table>태그
    • 데이터를 표로 만들어 주는 태그
    • <tr> : 테이블 행 선언
    • <th> : 테이블 첫 행의 제목줄 선언. 가운데 정렬과 굵게 표시가 기본값
    • <td> : 테이블 데이터(한 칸의 값)
    • border="숫자"로 표의 굵기
    • cellspacing="숫자"로 표의 경계값 빈 칸 지정
    • rowspan="숫자"만큼의 표의 행을 병합
    • colspan="숫자"만큼의 표의 열을 병합
